# منصة متابعة مؤشرات الاعتماد المدرسي

واجهة عربية حديثة لقياس مؤشرات الاعتماد المدرسي بطريقة عملية.

## المزايا
- تصميم احترافي بخطوط **Cairo** و **Tajawal**.
- هيكل عرض ثلاثي الطبقات: **المجالات → المعايير → المؤشرات**.
- قوائم تحقق تفاعلية لكل مؤشر (الشواهد + الوثائق).
- تقييم تلقائي لمستوى تحقق المؤشر حسب عدد الأدلة المتوفرة:
  - 🔴 منخفضة (2 فأقل)
  - 🟠 متوسطة (3-4)
  - 🟢 مرتفعة (5)
  - 🔵 مرتفعة جداً (6 فأكثر)
- تحديث حالة الإنجاز (لم يبدأ / قيد التنفيذ / مكتمل) مع لوحة مؤشرات فورية.
- حفظ البيانات محليًا في المتصفح (LocalStorage).
- تصدير نسخة احتياطية احترافية (JSON) واستعادتها لاحقًا على أي جهاز أو مشاركتها مع أعضاء الفريق للمتابعة من نفس نقطة التقدم.
- ربط المجلدات العامة في Google Drive وفحص اكتمال ملفات المؤشرات تلقائيًا وفق نظام تسمية موحد (`1-1-1-1-a-1.pdf` / `1-1-1-1-b-1.pdf`) مع تقرير نقص/اكتمال.

## فحص Google Drive
1. افتح المنصة ثم من قسم **فحص ملفات Google Drive** ألصق رابط المجلد (أو معرف المجلد فقط).
2. أدخل **Google API Key** فعّال لخدمة **Google Drive API**.
3. اضغط **بدء الفحص** ليتحقق النظام من:
   - وجود تسلسل المجلدات (المجال ← المعيار ← المؤشر).
   - وجود ملفات PDF المطلوبة لكل مؤشر.
   - صحة صيغة أسماء الملفات ومطابقتها للمؤشر.
